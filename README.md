# firebase-template

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### .env
```

VUE_APP_FIREBASE_PW="your password"
```
